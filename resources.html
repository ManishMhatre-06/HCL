Resources

Access all essential tools and documentation required to get started with Human-Centric Language (HCL). Download the language runtime, compiler, and official documentation below.

What’s Included

  • HCL Language: Runtime environment for executing natural language programs
  • HCL Compiler: Converts HCL instructions into executable Python code
  • Documentation: Complete guide covering installation, syntax, and usage

Getting Started

Start by downloading the HCL Language and Compiler. Once installed, follow the documentation to begin writing and executing programs using natural language instructions.
